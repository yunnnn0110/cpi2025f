2. 一元二次方程式



一元二次方程式，ax^2 + bx + c = 0，輸入a, b, c, 求 方程式的兩個實根。
  輸入說明
    第一個數(int) a

    第二個數(int) b

    第三個數(int) c

  輸出說明
    第一個實根 x1 = ((-b)+math.sqrt(b*b-4*a*c))/(2*a)

    第二個實根 x2 = ((-b)-math.sqrt(b*b-4*a*c))/(2*a)

    x1, x2 輸出到小數點第一位

    print("%.1f" %x1)

Tips



  要使用平方根function的話
  要載入math函式庫 

                                import math
                                math.sqrt()

解答



  引入math函式庫
  設定好輸入(int)和變數名
  將公式解程式化

                        //引入math函式庫
                        import math
                        //設定好輸入
                        a = int(input())
                        b = int(input())
                        c = int(input())
                        //計算公式解
                        root1 = ((-b)+math.sqrt(b*b-4*a*c))/(2*a)
                        root2 = ((-b)-math.sqrt(b*b-4*a*c))/(2*a)
                        //輸出答案
                        print("%.1f"%root1)
                        print("%.1f"%root2)
