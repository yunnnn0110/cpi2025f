6. 一元二次方程式



  一元二次方程式，aX^2 + bx + c = 0，輸入a, b, c, 求方程式的兩個實/虛根。
    T = math.sqrt(b*b-4*a*c)

    第一個根，x1 = (-b+T)/(2*a)
    第二個根，x2= (-b-T)/(2*a)

    本題需 import math

輸入輸出說明


  輸入說明
    第一行，輸入a (Integer)
    第二行，輸入b (Integer)
    第三行，輸入c (Integer)

  輸出說明
    1.當T>=0，輸出x1, x2為實根，輸出到小數點第一位
      第一行，輸出 print("%.1f” %(x1))
      第二行，輸出 print("%.1f” %(x2))
    2.當T<0，輸出x1 , x2為虛根，輸出到小數點第一位
      第一行，輸出 print("%.1f+%.1fi" %(x11,x12))
      第二行，輸出 print("%.1f-%.1fi" %(x21,x22))

                      import math
                      def func(a,b,c):

                          is_virtual = False

                          result = b*b-4*a*c

                          if(result < 0):
                              is_virtual = True
                              result = result * (-1)


                          if(is_virtual):
                              x_real = -b / (2*a)
                              x_virtual = math.sqrt(result)/(2*a)
                              print("%.1f+%.1fi" %(x_real,x_virtual))
                              print("%.1f-%.1fi" % (x_real, x_virtual))
                          else:
                              x_ans1 = (-b+math.sqrt(result))/(2*a)
                              x_ans2 = (-b-math.sqrt(result))/(2*a)
                              print("%.1f"%(x_ans1))
                              print("%.1f"%(x_ans2))
