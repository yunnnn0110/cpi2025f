4. BMI計算



  計算BMI = 體重(公斤) / (身高*身高)(公尺)
  輸入說明
    1.身高(float, 公尺)

    2.體重(Integer, 公斤)

  輸出說明
    1.結果須輸出到小數點第二位，print(format(bmi, '.2f'))

    2.第三位按照四捨六入五看偶規則進行數字簡化

  四捨六入五看偶規則:

    保留位數的後一位如果是5，要根據應看尾數「5」的前一位決定是捨去還是進入，如果是奇數則進位，如果是偶數則捨去。例如5.215保留兩位小數為5.22，而 5.225保留兩位小數為5.22。

解答



  設定好main函式和輸入

                      def BMI(height, weight):
                        # 計算BMI公式
                        bmi = weight/(height*height)
                        # 四捨六入五看偶規則
                        if (int)(bmi*1000 % 10) > 5:
                            bmi += 0.01
                        elif (int)(bmi*1000 % 10) == 5:
                            if (int)(bmi*100) % 2 == 1:
                                bmi += 0.01
                        # 最後把結果格式轉成只有小數點後兩位
                        bmi = (int)(bmi * 100) / 100
                        print(format(bmi, '.2f'))

                      def main():
                          height = float(input())
                          weight = int(input())
                          BMI(height, weight)
                      
                      if __name__ == "__main__":
                          main()
