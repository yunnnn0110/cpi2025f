18. 圖形繪製



  請使用迴圈繪製4種不同的圖形
  圖形請參考範例測資


  菱形:兩對角線長度為N且由”*”構成的菱形

  魚形:身體由"*"組成，是一個對角線長度皆為 N 的菱形，尾巴由"-"組成三角形，三角形的最長邊(底邊)在右，頂端朝左，是一個底為(N-2)，高為((N-1)/2)的三角形

    

輸入輸出說明


  輸入說明
    第一行，輸入整數 C，1 <= C <=4，代表接下來要畫的圖形種類
      C=1,為第一種圖形 正三角形 (參考 範例輸出 1)

      C=2,為第二種圖形 倒三角形 (參考 範例輸出 2)

      C=3,為第三種圖形 菱形 (參考 範例輸出 9)

      C=4,為第四種圖形 魚形 (參考 範例輸出 13)

    第二行，輸入整數 N ，0 < N < 50，代表這個圖形有N行
  輸出說明
    若N為奇數且3<=N<=49。根據輸入，畫出對應的圖形

    若N為偶數或者N<=2或N>=50。輸出 error

  再使用迴圈執行一行的funcution完成圖型的繪製

解答




                        # 畫一行#
                        def printSharp(n):
                            for i in range(n):
                                print('#', end='')
                        
                        # 畫一行*
                        def printStar(n):
                            for i in range(n):
                                print('*', end='')
                        
                        # 畫一行空格
                        def printSpace(n):
                            for i in range(n):
                                print(' ', end='')
                        
                        # 畫一行-
                        def printLine(n):
                            for i in range(n):
                                print('-', end='')
                        
                        # 畫三角形
                        def printTriangle(n):
                            for i in range(1, n+1):
                                printSharp(n-i)
                                printStar(i*2-1)
                                printSharp(n-i)
                                print()
                        
                        # 畫倒三角形
                        def printInvertedTriangle(n):
                            for i in range(n):
                                printSharp(i)
                                printStar(n*2 - i*2 - 1)
                                printSharp(i)
                                print()
                        
                        # 畫菱形
                        def printDiamond(n):
                            for i in range(1, n+1, 2):
                                printSpace(int((n-i)/2))
                                printStar(i)
                                print()
                            for i in range(n-2,0,-2):
                                printSpace(int((n-i)/2))
                                printStar(i)
                                print()
                        
                        # 畫魚形
                        def printFish(n):
                            for i in range(1, n+1, 2):
                                printSpace(int((n-i)/2))
                                printStar(i)
                                printSpace(n-i)
                                printLine(i//2)
                                print()
                            for i in range(n-2,0,-2):
                                printSpace(int((n-i)/2))
                                printStar(i)
                                printSpace(n-i)
                                printLine(i//2)
                                print()
                        
                        def main():
                            choose = input()
                            n = int(input())
                            if n % 2 == 0 or n < 3 or n > 49:
                                print('error')
                                return
                            if choose == '1':
                                printTriangle(n)
                            elif choose == '2':
                                printInvertedTriangle(n)
                            elif choose == '3':
                                printDiamond(n)
                            elif choose == '4':
                                printFish(n)
                        
                        main()
