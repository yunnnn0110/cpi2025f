10. 保齡球



  小明到保齡球館打保齡球,一輪有十局,假設小明一到八局都拿零分,剩下最後兩局
  ，試算總分，保齡球規則如下:

      (1)每局有10個保齡球瓶，擊倒1個球瓶得1分

      (2)每局最多有兩次投球機會

      (3)當局如果第一球將10個保齡球瓶全部擊倒(稱為strike)，當局只有一
      次投球機會，此局計分除了10分外，再加上後面兩球擊倒的球瓶數。

      (4)當局如果第一球未將10個保齡球瓶全部擊倒，但加上第二球則將10
      個保齡球瓶全部擊倒(稱為spare)，此局計分除了10分外，再加上後面
      一球擊倒的球瓶數。

      (5)當局若兩球都無法將球瓶全部擊倒，
      此局計分為兩球擊倒的球瓶數。

10. 保齡球



      (6)第10局如果第一球將球瓶全部擊倒，後面還有2次投球機會。

      (7)第10局如果第二球才將球瓶全部擊倒，後面還有1次投球機會。

      (8)第10局若兩球都無法將球瓶全部擊倒，則結束比賽。

      (9)總分為10局分數總合。

輸入輸出說明


  輸入說明
    第9局第一球分數 (int,0~10)

    第9局第二球分數 (上一球為10則省略) (int,和上一球總和最多為10)

    第10局第一球分數 (int,0~10)

    第10局第二球分數 (上一球為10則省略) (int,和上一球總和最多為10)

    第一次額外投球分數(第10局投球總分為10才輸入) (int,0~10)

    第二次額外投球分數(第10局第一球為10才輸入) (int,0~10)

  輸出說明
    總分 (int)

輸入輸出範例解釋


  輸入範例說明

    5(第9局第一球)
    5(第9局第二球)
    10(第10局第一球)
    0(因為第10局第一次就全倒，第一次額外投球)
    8(因為第10局第一次就全倒，第二次額外投球)

  輸出範例說明

    38(第9局兩次全倒，分數為5+5+10=20。第10局一次全倒，分數為10+0+8=18。總分為20+18=38)

                        #Spare判斷
                        def spare(ball1,ball2):
                            if(ball1+ball2)==10:
                                return True
                            return False
                        
                        #Strike判斷
                        def strike(ball):
                            if ball==10:
                                return True
                            return False
                        
                        #玩一局
                        def oneround():
                            score=0
                            ball1=int(input())
                            score+=ball1
                            if strike(ball1):
                                return 2,ball1,-1,score
                            ball2=int(input())
                            score+=ball2
                            if spare(ball1,ball2):
                                return 1,ball1,ball2,score
                            return 0,ball1,ball2,score
                        
                        #額外投球機會
                        def bonustime(roun10_state):
                            if roun10_state==1:
                                ball1=int(input())
                                return ball1,-1
                            elif roun10_state==2:
                                ball1=int(input())
                                ball2=int(input())
                                return ball1,ball2
                            return 0,0
                        
                        #額外加分
                        def addAnotherPoint(ball1,ball2,remainTime,addedround_point):
                            if(ball2==-1):
                                if(remainTime>0):
                                    addedround_remainaddedtime,addedround_score= remainTime-1,ball1+addedround_point
                                else:
                                    addedround_remainaddedtime,addedround_score=0,addedround_point
                            else:
                                if(remainTime==2):
                                    addedround_remainaddedtime,addedround_score= 0,ball1+ball2+addedround_point
                                elif(remainTime==1):
                                    addedround_remainaddedtime,addedround_score= 0,ball1+addedround_point
                                else:
                                    addedround_remainaddedtime,addedround_score=0,addedround_point
                            
                            return addedround_remainaddedtime,addedround_score
                        
                        
                        def startGame():
                            #初始化
                            all_state = [0]*2
                            all_score=[0]*2
                            
                            # all_state 
                            # 0 normal
                            # 1 spare
                            # 2 strike
                            #第九局
                            all_state[0], ball1, ball2, all_score[0] = oneround()
                            #第十局
                            all_state[1], ball1, ball2, all_score[1] = oneround()
                            #第九局額外得分
                            all_state[0], all_score[0] = addAnotherPoint(ball1, ball2, all_state[0], all_score[0])
                            #額外投球機會
                            ball1,ball2=bonustime(all_state[1])
                            #第十局額外得分
                            all_state[1], all_score[1] = addAnotherPoint(ball1, ball2, all_state[1], all_score[1])
                            #第九、十局Strike的第九局額外得分
                            all_state[0], all_score[0] = addAnotherPoint(ball1, ball2, all_state[0], all_score[0])
                            return all_score[0]+all_score[1]
                        
                        def main():
